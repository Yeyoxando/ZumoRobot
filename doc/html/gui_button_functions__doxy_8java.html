Detailed Description

Manage the buttons functionality.
(Real name -> guiButtonFunctions.pde).

Details.
Some buttons does different things depending if zumo is on manual or autonomous mode. You can find the descriptions below.
Manage the buttons functionality as gui file is autogenerated and delete comments and everything.
They have to be done here to use doxygen as it will be deleted in the other file.
